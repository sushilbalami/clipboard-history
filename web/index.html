What it does

Built for people who live on copy and paste.

  • Global shortcut to open history from anywhere
  • Searchable clipboard list with timestamps and source apps
  • One-tap pin, delete, or copy actions
  • Optional auto-paste flow with Accessibility permission
  • Launch at login and adjustable history size

Install

Download the latest release, then drag the app into Applications.

Every push to main publishes a fresh macOS build to GitHub Releases. Download the latest ZIP, extract it, move Clipboard History.app into /Applications, then grant Accessibility if you want auto-paste.
