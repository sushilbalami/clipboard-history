What it does

Built for people who live on copy and paste.

  • Global shortcut to open history from anywhere
  • Searchable clipboard list with timestamps and source apps
  • One-tap pin, delete, or copy actions
  • Optional auto-paste flow with Accessibility permission
  • Launch at login and adjustable history size

Install

Install the app on your Mac in one command.

After cloning or downloading this project, run the installer from the repository root. The script builds a release binary, creates the app bundle, and copies it into your Applications folder. 

./scripts/install-app.sh
01

Prepare macOS

Use a Mac running macOS 14 or later with Xcode Command Line Tools available for Swift builds.

02

Run installer

Execute ./scripts/install-app.sh from the project root to build and install the app.

03

Grant permission

Open the app, enable Accessibility if you want auto-paste, and set your preferred global shortcut.
